# AutoApply AI 🚀

AI-powered job application assistant that analyzes resumes and job descriptions to generate match scores, cover letters, and HR emails.

## Features
- Resume analysis
- Job match scoring
- Missing skills detection
- Cover letter generation
- HR email generation

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Ollama (Local AI)

## How to Run
1. Install dependencies
2. Run server: node server.js
3. Open http://localhost:5001
